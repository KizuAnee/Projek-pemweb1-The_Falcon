Bantuan

  1. Pertama-tama, buat akun dengan menekan tombol Get Started atau Sign Up
  2. Jika Anda sudah memiliki akun, harap masuk dengan metode Login
  3. Pada Pop Up Input Data User, harap masukkan data umur, tinggi, berat, jenis kelamin, dan program yang akan Anda ikuti
  4. Selanjutnya, Anda akan diarahkan untuk login. Harap memasukkan Username dan Password yang sesuai
  5. Selesai! Semoga Anda konsisten mengikuti program ini!
